```mermaid
sequenceDiagram
    autonumber
    actor Player as 플레이어
    participant UI as 길드가입_화면
    participant Battle as 전투
    participant Pl as 플레이어
    participant Guild as 길드
    
    Player->>+UI: 길드가입 요청(플레이어id, 길드명)
    UI->>+Battle: 길드가입(플레이어id, 가입캐릭터, 가입길드)
    
    %% 플레이어 체크 (Include)
    Battle->>+Pl: 플레이어체크(플레이어id)
    Pl-->>-Battle: true (검증 통과)
    
    %% Aggregation 관계에 따른 길드 가입 요청 (boolean 반환)
    Note over Battle, Guild: 길드 "1" o-- "*" 캐릭터 (Aggregation)
    Battle->>+Guild: 캐릭터가입(가입캐릭터)
    Guild-->>-Battle: 가입 결과 반환 (boolean)
    
    %% boolean 결과에 따른 조건 분기 (최대 인원 5명 제한)
    alt 가입 결과 == true (정원 내)
        Battle-->>UI: 가입 성공 메시지 (String)
        UI-->>Player: 길드에 가입되었습니다.
    else 가입 결과 == false (정원 초과)
        Battle-->>UI: 가입 실패 메시지 (String)
        UI-->>Player: 길드 정원이 가득 차 가입할 수 없습니다.
    end
    
    deactivate Battle
    deactivate UI
