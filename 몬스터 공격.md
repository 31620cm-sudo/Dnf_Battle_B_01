```mermaid

sequenceDiagram
    autonumber
    actor Player as 플레이어
    participant UI as Attack_Monster_UI
    participant Battle as 전투
    participant Pl as 플레이어(Entity)
    participant Ch as 캐릭터

    Player->>+UI: 몬스터 공격 요청(플레이어id, 공격자)
    UI->>+Battle: 몬스터공격(플레이어id, 공격자)
    
    %% 플레이어 체크 (Include 관계)
    Battle->>+Pl: 플레이어체크(플레이어id)
    Pl-->>-Battle: 검증 결과 (true/false)
    
    alt 플레이어체크 성공 (true)
        %% 직업별 스킬 발동 및 데미지 계산
        Battle->>+Ch: 스킬발동()
        
        alt 공격자가 "전사"인 경우
            Note over Ch: "검 휘두르기" 발동<br/>데미지 = 공격력 * 1.5
        else 공격자가 "마법사"인 경우
            Note over Ch: "파이어볼" 발동<br/>데미지 = 공격력 * 2.0
        end
        
        Ch-->>-Battle: 계산된 데미지 반환 (float)
        
        %% 데미지 등급 부여
        alt 데미지 >= 200
            Note over Battle: 등급 = "S급 공격"
        else 데미지 >= 100
            Note over Battle: 등급 = "A급 공격"
        else 데미지 < 100
            Note over Battle: 등급 = "B급 공격"
        end
        
        Battle-->>-UI: 공격 결과 반환 (등급 및 데미지 정보)
        UI-->>-Player: 공격 성공 및 등급 출력
    else 플레이어체크 실패 (false)
        Battle-->>-UI: 공격 실패 (권한 없음)
        UI-->>-Player: "공격 권한이 없습니다."
    end
