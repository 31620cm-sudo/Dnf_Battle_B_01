```mermaid

sequenceDiagram
    autonumber
    actor 플레이어 as 플레이어
    participant UI as 몬스터_공격_UI
    participant 컨트롤러 as 전투
    participant 유저 as 플레이어_엔티티
    participant 내캐릭터 as 캐릭터

    플레이어->>UI: 몬스터 공격 요청 (플레이어id, 공격자: 캐릭터)
    activate UI
    UI->>컨트롤러: 몬스터공격(플레이어id: String, 공격자: 캐릭터)
    activate 컨트롤러

    %% 1. 플레이어 체크
    컨트롤러->>유저: 플레이어체크(플레이어id: String)
    activate 유저
    유저-->>컨트롤러: 결과 반환 (true / false)
    deactivate 유저

    %% 2. 조건문 (플레이어 체크 성공 시)
    alt 플레이어id == "hero" (검증 성공)
        
        %% 직업에 따른 다형성 스킬 발동
        컨트롤러->>내캐릭터: 스킬발동()
        activate 내캐릭터
        
        alt 직업이 전사인 경우
            Note over 내캐릭터: 스킬: "검 휘두르기"<br/>대미지 = 공격력 * 1.5
        else 직업이 마법사인 경우
            Note over 내캐릭터: 스킬: "파이어볼"<br/>대미지 = 공격력 * 2.0
        end
        
        내캐릭터-->>컨트롤러: 대미지 반환 (대미지: float)
        deactivate 내캐릭터

        %% 내부 로직: 등급 부여
        컨트롤러->>컨트롤러: +등급부여(대미지: float)
        Note over 컨트롤러: 대미지 >= 200: S급<br/>대미지 >= 100: A급<br/>대미지 < 100: B급
        컨트롤러-->>컨트롤러: -등급 반환 (String)

        컨트롤러-->>UI: 공격 결과 반환 (대미지, 등급)
    else 검증 실패
        컨트롤러-->>UI: 공격 실패 (권한 없음)
    end
    deactivate 컨트롤러

    UI-->>플레이어: 공격 결과 및 등급 화면 표시
    deactivate UI
